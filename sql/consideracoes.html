1.12. Considerações Finais

Nesse Capítulo, foi feita uma breve introdução aos conceitos e funcionalidades presentes em um servidor típico de bancos de dados, usando o PostgreSQL para trabalhar no domínio geoespacial. Para mais detalhes sobre o tema de Bancos de Dados Geográficos e uso do PostgreSQL no domínio geoespacial, consulte [11].

Nesse livro online é apresentado um histórico do desenvolvimento tecnológico dos sistemas de bancos de dados e sua integração com universo geoespacial (Introdução a Sistemas de Bancos de Dados). No Capítulo 2 - Sistemas de Bancos de Dados Relacionais são apresentadas seções mais detalhadas sobre a linguagem SQL e suas diversas cláusulas, bem como as estratégias para construção de consultas. O Capítulo 3 - Tipos de Dados e Operações Espaciais apresenta em detalhes uma discussão sobre os elementos geométricos e, principalmente, o formalismo dos operações topológicos usados nos relacionamentos espaciais. Esse capítulo ainda contem diversos exemplos de consultas espaciais para compreensão dos conceitos aplicados. O Capítulo 4 - Estruturas de Dados e Métodos de Acesso Multidimensionais discute um tópico importante para construção de grandes bases de dados, que é a utilização de métodos de indexação para acelerar a execução das consultas. Finalmente, o Capítulo 5 - Algoritmos Geométricos discute sobre as idéias subjacentes à implementação dos operadores espaciais.
