3.1.4. Instrumento de Crédito

Instrumento de Crédito é a denominação de um instrumento específico que formaliza as operações celebradas com o banco, compreendidos, nesse conceito, os títulos de crédito e os contratos, ao qual aderem os demais documentos a ele vinculados. Os instrumentos de crédito mais conhecidos são hipoteca, nota promissória e a duplicata. O arquivo InstrumentoCredito.csv contém os registros dos tipos de instrumentos de créditos utilizados no financiamento agrícola. Sua estrutura pode ser vista na Tabela 3.3.

Tabela 3.3 - Estrutura do arquivo contendo a lista dos instrumentos de crédito.

nome

tipo

descrição

PK

cd_inst_credito

char(2)

Código de Instrumento de Crédito

descricao

text

Nome do concedente do Crédito

sigla

text

Sigla de Instrumento de Crédito

A coluna cd_inst_credito contém um código único para cada instrumento de crédito, sendo a chave primária dos registros desse conjunto de dados. Essa coluna é utilizada para relacionar os instrumentos de crédito aos contratos, como será visto na Seção 3.1.9 - Operações Contratadas com Recursos Públicos e Privados.

Nota

O arquivo InstrumentoCredito.csv encontra-se carregado na tabela instrumentocredito do banco de dados PostgreSQL.

3.1.4.1. Consultas

Consulta 1. Listar os tipos de instrumentos de crédito.

Solução:
  SELECT *
    FROM instrumentocredito
ORDER BY descricao;
