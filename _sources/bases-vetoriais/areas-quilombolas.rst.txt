Áreas Quilombolas
=================


As comunidades quilombolas são grupos étnicos – predominantemente constituídos pela população 
negra rural ou urbana –, que se autodefinem a partir das relações específicas com a terra, 
o parentesco, o território, a ancestralidade, as tradições e práticas culturais próprias. 
Por força do Decreto nº 4.887, de 2003, o INCRA (Instituto Nacional de Colonização e Reforma 
Agrária) é a autarquia competente, na esfera federal, pela titulação dos territórios quilombolas. 
As terras ocupadas por remanescentes das comunidades dos quilombos são aquelas utilizadas 
para a garantia de sua reprodução física, social, econômica e cultural. 


As áreas quilombolas utilizados nesse curso foram baixadas, em arquivos no formato shapefile, 
do `Acervo Fundiário <https://acervofundiario.incra.gov.br/acervo/login.php>`__ do INCRA e 
inseridos na tabela ``areas_quilombolas``. Um sumário de todas as tabelas vetoriais e suas principais 
características podem ser encontrado em  :ref:`Sumário das tabelas vetoriais <sec_bases_vetoriais_sumario>`.

Consultas
---------


**Consulta 1.** Recuperar o número de terras quilombolas.


.. collapse:: Solução:


    .. tabs::

        .. tab:: SQL

            .. code-block:: sql

                SELECT COUNT(*)
                  FROM areas_quilombolas;


**Consulta 2.** Recuperar o número de terras quilombolas por estado.


.. collapse:: Solução:


    .. tabs::

        .. tab:: SQL

            .. code-block:: sql

                SELECT cd_uf, count(*) 
                  FROM areas_quilombolas
                  GROUP BY cd_uf
                  ORDER BY count(*) DESC


**Consulta 3.** Recuperar informações sobre as geometrias das áreas quilombolas (nome da coluna, SRID e tipo geométrico).


.. collapse:: Solução:


    .. tabs::

        .. tab:: SQL

            .. code-block:: sql

                SELECT * 
                  FROM geometry_columns
                  WHERE f_table_name = 'areas_quilombolas'   


**Consulta 4.** Selecionar as glebas (a partir de 2021) que tem interseção com áreas quilombolas. 


.. collapse:: Solução:


    .. tabs::

        .. tab:: SQL

            .. code-block:: sql

                SELECT gle.ano, gle.gid
                  FROM  areas_quilombolas as qui, sicor_glebas as gle
                  WHERE ST_INTERSECTS(
		              ST_Transform(qui.geom, 4674), gle.geom)
		              AND ano > 2020
                  ORDER BY gle.ano, gle.gid DESC
